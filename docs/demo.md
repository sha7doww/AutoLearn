### 第二部分：系统演示（12分钟）

#### 演示1: 知识状态分析（4分钟）

**页面**: Dashboard

**步骤**:

1. **展示界面**（30秒）
   ```
   "这是我们的智能导学系统主界面。左侧是成绩输入区，
   我们已经预填了8门基础课程的示例数据。"
   ```

2. **分析知识状态**（30秒）
   - 点击"分析知识状态"按钮
   - 等待计算结果

3. **解释结果**（3分钟）
   - **综合水平**: "系统计算出这位学生的综合水平是78分"
   - **雷达图**: "这个雷达图展示了8个知识域的掌握情况"
   - **优势领域**: "系统识别出编程基础和算法基础是优势领域"

**讲解重点**:
- IRT模型原理
- 多维度知识分析
- 可视化效果

#### 演示2: 智能课程推荐（4分钟）

**页面**: Dashboard（继续向下）

**步骤**:

1. **展示推荐结果**（30秒）
   ```
   "知识状态分析完成后，系统自动生成了个性化推荐。
   可以看到推荐了5门课程。"
   ```

2. **解释推荐详情**（2分钟）
   - **匹配度**: "每门课程都有匹配度评分"
   - **个性化难度**: "注意这里的难度是个性化的"
   - **推荐理由**: "系统自动生成了推荐理由"
   - **先修检查**: "自动检查先修要求是否满足"

3. **对比演示**（1.5分钟）
   ```
   "如果我修改一下数学成绩...
   ```

## 🔧 常见问题处理

### 问题1: 系统无响应

**原因**: 后端未启动

**解决**:
```bash
# 检查后端
curl http://localhost:8000/health

# 重启
scripts/stop.sh
scripts/start.sh
```

### 问题2: 雷达图不显示

**原因**: ECharts CDN加载失败

**解决**: 刷新页面，或准备截图备用

### 问题3: 前端加载慢

**原因**: Node.js开发服务器

**解决**: 提前10分钟启动系统预热

## 🎬 演示脚本

### 开场白

```
各位老师、同学，大家好！

我今天要展示的是SmartPath智能导学系统。
这是一个基于知识追踪的个性化学习路径规划平台。

系统的核心创新点是：
1. 使用IRT模型进行知识状态追踪
2. 个性化难度评估和课程推荐
3. 智能学习路径规划

下面我将通过实际演示来展示系统的功能。
```

### 知识状态分析

```
首先是知识状态分析功能。

这里我输入了一位学生8门基础课程的成绩。
点击"分析知识状态"。

系统使用IRT模型进行计算...
可以看到，系统给出了：
1. 综合水平78分
2. 雷达图显示各知识域掌握情况
3. 识别出编程基础和算法基础是优势领域

这是基于Item Response Theory的科学计算结果。
```

### 智能推荐

```
知识状态分析完成后，系统自动生成个性化推荐。

这里推荐了"算法分析与设计"，匹配度85%。
注意看推荐理由：满足先修要求、知识储备充分、难度适中。

这里的"难度适中"是个性化的。
如果我降低数学成绩...
（演示修改）
推荐结果就会变化，难度评估也不同了。

这就是个性化推荐的核心价值。
```

### 学习路径规划

```
接下来演示学习路径规划。

假设这位学生想学"人工智能"。
系统会：
1. 检查先修要求
2. 过滤已完成课程
3. 生成最优学习序列

可以看到时间线展示了5门课的学习路径，
总共15学分，预计1个学期完成。

每门课都标注了先修关系和推荐理由。
```

```
最后简单展示技术架构。

（切换到API文档）
这是Swagger自动生成的API文档。
后端使用FastAPI框架，支持RESTful API。

我们实现了知识追踪、推荐、路径规划等核心接口。
前端使用Vue 3和Three.js，提供良好的用户体验。

（切换到3D图谱）
这是67门课程的3D知识图谱，
可以交互式浏览课程关系网络。
```

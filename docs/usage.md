## 📊 功能详解

### 1. 知识状态分析

**路径**: Dashboard → 知识状态分析

**功能**: 基于IRT模型分析学生知识掌握程度

#### 使用步骤

1. **输入成绩**
   - 在"输入您的课程成绩"区域
   - 为8门基础课程输入分数(0-100)
   - 系统已预填示例数据

2. **分析知识状态**
   - 点击"分析知识状态"按钮
   - 等待计算(约1-2秒)

3. **查看结果**
   - **综合水平评分**: 0-100分
   - **知识掌握度雷达图**: 多维度可视化
   - **优势领域**: 掌握度>70%的知识域
   - **薄弱环节**: 掌握度<40%的知识域
   - **详细分析**: 每个知识域的具体掌握度

#### 示例数据

```
高等数学: 85
线性代数: 78
概率论与数理统计: 82
程序设计原理与C语言: 90
数据结构: 88
离散数学: 75
计算机组成与实践: 80
操作系统: 85
```

#### 结果解读

- **综合水平 ≥ 80**: 基础扎实，可学习高级课程
- **综合水平 60-80**: 基础良好，需针对性提升
- **综合水平 < 60**: 建议加强基础课程

### 2. 智能课程推荐

**路径**: Dashboard → 智能推荐课程

**功能**: 基于知识状态个性化推荐课程

#### 触发方式

完成"知识状态分析"后自动显示推荐结果

#### 推荐信息

每个推荐包含：

- **课程名称**: 推荐的课程
- **匹配度**: 0-100%，越高越适合
- **难度评估**: 个性化难度（简单/中等/较难/困难）
- **推荐理由**: 自动生成的推荐说明
- **先修要求**: 是否满足先修条件
- **缺失课程**: 需要先学习的课程数量

#### 匹配度计算

```
匹配度 = 先修满足度(40%) + 知识准备度(30%) + 最优挑战(30%)
```

- **先修满足度**: 是否完成所有先修课程
- **知识准备度**: 知识储备与课程要求的契合度
- **最优挑战**: 难度是否在最佳学习区间

#### 个性化难度

同一门课程，不同学生看到的难度不同：

- **基础好的学生** → 难度降低
- **基础差的学生** → 难度增加

### 3. 学习路径规划

**路径**: Dashboard → 选择目标课程 → 生成学习路径

**功能**: 智能生成到目标课程的最优学习序列

#### 使用步骤

1. **选择目标课程**
   - 在"选择目标课程"下拉框中选择
   - 例如：人工智能、深度学习等

2. **生成路径**
   - 点击"生成学习路径"按钮
   - 系统计算最优路径

3. **查看结果**
   - **时间线可视化**: 学习顺序展示
   - **课程序列**: 第1门、第2门...
   - **总学分**: 需要完成的总学分
   - **预计学期**: 按每学期20学分计算

#### 时间线功能

- **课程状态**: 已完成/进行中/待学习
- **课程信息**: 难度、学分、类型
- **先修关系**: 显示每门课的先修要求
- **推荐理由**: 为什么推荐此课程

#### 交互功能

- **点击课程**: 查看详细信息
- **导出路径**: 下载JSON格式学习计划
- **调整路径**: 自定义学习顺序

### 4. 3D知识图谱

**路径**: http://localhost:8080/home

**功能**: 3D可视化课程关系网络

#### 操作方式

- **旋转视角**: 鼠标左键拖动
- **平移视角**: 鼠标右键拖动
- **缩放**: 鼠标滚轮
- **点击节点**: 查看课程详情

#### 搜索功能

1. 在顶部输入课程名称
2. 点击搜索
3. 相机自动聚焦到目标课程
4. 节点高亮显示

#### 图谱特点

- **67个课程节点**: 球形布局
- **150+先修关系**: 贝塞尔曲线连接
- **实时渲染**: Three.js驱动
- **流畅交互**: 60fps渲染

## 🔧 高级功能

### API直接调用

访问 http://localhost:8000/docs 查看完整API文档

#### 示例：计算知识状态

```bash
curl -X POST http://localhost:8000/api/knowledge/state \
  -H "Content-Type: application/json" \
  -d '{
    "student_id": "student123",
    "course_scores": {
      "高等数学": 85,
      "数据结构": 88
    }
  }'
```

#### 示例：获取课程推荐

```bash
curl -X POST http://localhost:8000/api/knowledge/recommend \
  -H "Content-Type: application/json" \
  -d '{
    "student_id": "student123",
    "knowledge_state": {
      "数学基础": 0.75,
      "编程基础": 0.85
    },
    "completed_courses": [1, 30, 37],
    "max_recommendations": 5
  }'
```

## ⚠️ 常见问题

### Q1: 点击"分析知识状态"没反应？

**原因**: 可能后端未启动或前端未连接

**解决**:
```bash
# 检查后端状态
curl http://localhost:8000/health

# 查看日志
tail -f logs/backend.log
```

### Q2: 雷达图不显示？

**原因**: ECharts CDN加载失败

**解决**: 检查网络连接，或刷新页面重试

### Q3: 推荐结果为空？

**原因**: 知识状态未计算

**解决**: 先完成"知识状态分析"

### Q4: 如何修改成绩？

直接在输入框中修改数值，然后重新点击"分析知识状态"

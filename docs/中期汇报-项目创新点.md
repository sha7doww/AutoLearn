# 项目创新点

本项目在智能导学系统领域提出并实现了以下创新方法：

## 一、知识感知的自适应搜索策略

传统推荐系统对所有学生使用相同的推荐策略，本项目提出**基于学生知识状态动态调整搜索策略**的方法。通过知识追踪模型量化学生的知识掌握情况后，系统自动调整推荐的深度和粒度：基础薄弱的学生获得更详细的先修课程建议和基础资料，基础扎实的学生直接推荐进阶内容。这种自适应机制使推荐更加个性化，避免了"一刀切"的问题。

## 二、Agentic Search架构的应用

本项目将近年来兴起的**智能体（Agent）技术**引入教育推荐领域。构建基于LangChain的ReAct Agent系统，能够根据学生问题自主调用多个专用工具（知识图谱查询、学习资料检索、学生状态分析等）进行多步推理，生成个性化学习建议。与传统的单次推荐不同，Agent系统具备推理能力和工具调用能力，能够解释推荐理由（如"因为你在X知识点掌握较好，可以直接学习Y课程"），增强了系统的可解释性和可信度。

## 三、知识追踪与推荐的深度融合

现有研究中，知识追踪（KT）和推荐系统通常是独立的模块。本项目创新性地将两者**深度融合**：知识追踪模块持续监测学生的学习进展，一旦检测到知识状态发生显著变化，系统自动触发推荐策略的重新规划，形成"学习 → 状态更新 → 重新推荐"的闭环。这种动态调整机制确保推荐始终符合学生的最新能力水平，而非基于过时的历史数据。

## 四、项目特色

**技术创新与工程实践结合**：本项目不仅提出新的算法思路，还注重工程实现和实际应用，构建了完整的前后端系统，为后续推广应用奠定基础。

**多学科交叉融合**：融合了知识追踪（教育技术学）、Agent系统（人工智能）、推荐算法（数据挖掘）、图数据库（数据库技术）等多个领域的先进方法，体现了跨学科研究的特点。
